# MAX Engine

The **MAX Engine** is a modular, GitHub‑native automation system designed to generate runs, modules, documentation, and expansion files entirely through GitHub Actions.  
It is built to be portable, self‑contained, and easy to extend with additional engine architectures.

## Features

- Fully automated execution via GitHub Actions  
- Generates run folders, module outputs, and documentation  
- Portable engine folder (`max-engine/`) for clean repo organization  
- Works entirely from mobile or desktop  
- Zero local setup required
